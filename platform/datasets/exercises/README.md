# Exercise Datasets

Canonical exercise bundles live under `datasets/exercises/<track>/...`.

Supported tracks:

- `piscine`
- `exams`
- `knr`
- `theory_to_practice`
- `bug_injection`
- `completion`
- `prediction`

Required bundle assets:

- `exercise.yml`
- `statement.md`
- `tests/tests.yml`
- `starter/`
- `reference/`
