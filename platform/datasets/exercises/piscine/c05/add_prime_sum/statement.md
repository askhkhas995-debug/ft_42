# add_prime_sum

Write a program that takes a positive integer as argument and displays the sum of all prime numbers inferior or equal to it followed by a newline.

If the number of arguments is not 1, or if the argument is not a positive number, just display 0 followed by a newline.

Yes, the exercises are strictly in C language, so you MUST write a function that formats int to chars using `write` instead of `printf`.

- Allowed functions: `write`, `exit`
